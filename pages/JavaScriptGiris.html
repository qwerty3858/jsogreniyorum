JavaScript Giriş

Bu sayfa da JavaScript hakkında temel bilgilerden bahsetmiş olacağım sizler için.

JavaScript Nedir?

JavaScript başlangıçta “web sayfalarını canlı hale getirmek” için oluşturuldu . Bu dildeki programlara komut dosyası denir . Doğrudan bir web sayfasının HTML'sine yazılabilir ve sayfa yüklendikçe otomatik olarak çalıştırılabilirler. Komut dosyaları düz metin olarak sağlanır ve yürütülür. Çalıştırmak için özel hazırlık veya derlemeye ihtiyaçları yoktur. Bu açıdan, JavaScript Java adı verilen başka bir dilden çok farklıdır .

-Neden Java Script olarak adlandırılıyor ? :


JavaScript oluşturulduğunda, başlangıçta başka bir adı vardı: “LiveScript”. Ancak Java o zamanlar çok popülerdi, bu yüzden yeni bir dili Java'nın “küçük kardeşi” olarak konumlandırmanın yardımcı olacağına karar verildi. Ancak geliştikçe, JavaScript ECMAScript adlı kendi belirtimiyle tamamen bağımsız bir dil haline geldi ve şimdi Java ile hiçbir ilişkisi yok.

Bugün, JavaScript yalnızca tarayıcıda değil, aynı zamanda sunucuda veya aslında JavaScript motoru adı verilen özel bir programa sahip herhangi bir cihazda da çalıştırılabilir . Tarayıcıda bazen “JavaScript sanal makinesi” adı verilen yerleşik bir motor bulunur. Farklı motorların farklı kod kodları vardır. Örneğin:

  • V8 - Chrome ve Opera'da.

  • SpiderMonkey - Firefox'ta.

  • … IE'nin farklı sürümleri için “Trident” ve “Chakra”, Microsoft Edge için “ChakraCore”, Safari için “Nitro” ve “SquirrelFish”
    gibi başka kod adları da vardır.




Özet

  • JavaScript başlangıçta yalnızca tarayıcı dili olarak oluşturulmuştur, ancak şimdi diğer birçok ortamda da kullanılmaktadır.

  • Bugün, JavaScript, HTML / CSS ile tam entegrasyon ile en yaygın olarak kabul edilen tarayıcı dili olarak benzersiz bir
    konuma sahiptir.

  • JavaScript'e "aktarılan" ve belirli özellikler sağlayan birçok dil vardır. JavaScript konusunda uzmanlaştıktan sonra en
    azından kısa bir süre onlara göz atmanız önerilir.
