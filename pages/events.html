Events(Olaylar)

Event Nedir?

JavaScript'in HTML ile etkileşimi, kullanıcı veya tarayıcı bir sayfayı değiştirdiğinde meydana gelen olaylar aracılığıyla gerçekleştirilir.
Sayfa yüklendiğinde buna olay adı verilir. Kullanıcı bir düğmeyi tıkladığında, bu tıklama da bir olaydır. Diğer örnekler arasında herhangi bir tuşa basma, pencereyi kapatma, pencereyi yeniden boyutlandırma vb. Gibi etkinlikler yer alır.
Geliştiriciler bu olayları, düğmelerin pencereleri kapatmasına, kullanıcılara görüntülenecek mesajlara, verilerin doğrulanmasına ve akla gelebilecek diğer tüm yanıt türlerine neden olan JavaScript kodlu yanıtları yürütmek için kullanabilirler.
Etkinlikler, Belge Nesne Modeli (DOM) Düzey 3'ün bir parçasıdır ve her HTML öğesi, JavaScript Kodunu tetikleyebilecek bir dizi etkinlik içerir.

onclick Event Türü

Bu, kullanıcı faresinin sol düğmesini tıklattığında oluşan en sık kullanılan olay türüdür. Bu olay türüne karşı doğrulama, uyarı vb. Koyabilirsiniz.
Örnek olarak
Aşağıdaki örneği deneyin.
< html> < head>
< script type = "text/javascript">
< !--
function sayHello() {
alert("Hello World")
}
//-->
< /script>
< /head>
< body>
< p>Click the following button and see result
< form>
< input type = "button" onclick = "sayHello()" value = "Say Hello" />
< /form>

onsubmit Event Türü

onsubmit , bir form göndermeye çalıştığınızda oluşan bir olaydır. Form doğrulamanızı bu olay türüne karşı koyabilirsiniz.
Misal
Aşağıdaki örnek, onsubmit'in nasıl kullanılacağını gösterir. Burada , web sunucusuna bir form verisi göndermeden önce bir validate () işlevi çağırıyoruz . Eğer validate () , form sunulacaktır işlevini verir true, aksi takdirde veri göndermek olmaz.
Aşağıdaki örneği deneyin.

< html>
< head>
< script type = "text/javascript">
< !--
function validation() {
yapılacak işlemler ...
}
//-->
< /script>
< /head>
< body>
< form method = "POST" action = "t.cgi" onsubmit = "return validate()">
.......
< input type = "submit" value = "Submit" />
< /form>
< /body>
< /html>

onmouseover ve onmouseout Event Türü

Bu iki etkinlik türü, görüntülerle ve hatta metinle de güzel efektler oluşturmanıza yardımcı olacaktır. Onmouseover olay tetikleyici herhangi elemanın üzerinde fareyi getirdiğinizde ve onmouseout tetikleyicileri bu öğeden farenizi dışarı taşıdığınızda. Aşağıdaki örneği deneyin.
< html>
< head>
< script type = "text/javascript">
< !--
function over() {
document.write ("Mouse Over");
} function out() {
document.write ("Mouse Out");
}
//-->
< /script>
< /head>
< body>
< p>Sonucu görmek için farenizi bölümün içine getirin:


< div onmouseover = "over()" onmouseout = "out()">
< h2> Bu bölümün içinde
< /div>
