Koşul Yapıları

Günlük hayatta hepimiz içinde eğer(if) geçen cümleler kurmuşuzdur.(bağlaç olan eğer :) )
Eğer sınavı geçersem kutlama yapacağım.
Eğer html öğrenirsen sonra css öğren.
Görüldüğü gibi "kişinin kutlama yapmasi sinavi geçmesine bağlı"
Css öğrenmesi "Html biliyor olmasina bağlı"
Burdaki şartlari sağlamazsa amacina ulaşamaz.şimdi javascript dilinde olaylara bakalim

Örneklerle Daha iyi Anlayalım...

< script>
var ortalama=85;
if(ortalama > 45){
alert("Ogrenci Dersi Gecti");
}



Neler oldu yazalim.
Ogrencinin ortalamasi 85,ortalama isimli degiskene atanadi
if(kosul) soldaki yapi geregi javascript icerdeki kosulu degerlendirir
kosulumuz ortalama > 50 bu kosul boolean turunde bir deger dondurur
demistik,yani true yada false
kosul=85 > 45 kosulu geriye true döndürür
son durum if(true)
kosul true oldugu icin if blogu calisacaktir. (süslü parantezler içindeki kodlar)
kosul geriye false dondurseydi if blogu hic calismayacakti
ortalamayi 40 yap ve tekrar test et , calismadigni gor

Yukardaki örneğin eksik tarafi , eğer öğrenci başarısız olduysa hiç mesaj vermemesiydi,şimdi bu sorunu çözerek else kavramini öğrenelim.

if-else

< script>
var ortalama=25;
if(ortalama > 45){
alert("Ogrenci Dersi Gecti");
}else{ alert("Ogrenci Dersten Kaldı!");
}


Neler oluyor inceleyelim:
if(kosul) sisteminde bu defa kosul=false (25>45 karsilastirmasi false dondurur)
Dolayisiyla if blogu hic calismadan else blogu calisacaktir
Kisaca if kosulu dogruysa ilk blok,değilse ikinci blok calısır

Şimdi ise ogrencinin notu konusunda degisik olasiliklari da hesaba katiyoruz,her blok icin alert mesaji yazdirmamak icin daha esnek bir yontem kullanacagim!.

< script>
var ortalama=25;
var mesaj=""; //not durumuna gore mesaji saklayacak degisken
//yukardaki ornekler 200 gibi deger icinde true donduruyordu
//bu if yapisi bunu engelliyor cunku 2 adet kosul sonucuna gore blok calisacak
//&& operatoru geregi 2 kosulda true dondurmeli ki sonuc true olabilsin
if(ortalama>=45 && ortalama<=100){
mesaj="Ogrenci Dersi Gecti";
//Bu blok calistiysa ogrencinin notlari 45e esit yada buyuk
//Ogrencinin notu 100e esit yada kucuk
}else{ mesaj="Ogrenci Dersten Kaldı";
//Bu blok calistiysa ogrencinin notu:
//45den kucuk olabilir (negatif sayilar bile)
//100den de buyuk olabilir!
//if kosulu saglanmadigi icin burasi otomatik calisir unutma
}
alert(mesaj);
< /script>


Javascript Koşullu Son Örneğimiz

Evet if else kod yapisini tamamlamak icin bu son baslik, Her adimda kodu daha yararli yapmamiza ragmen , bazi eksikler vardi simdi onlari düzeltelim. Ogrencinin notuna göre "Pekiyi","iyi","orta","gecer","zayıf" gibi mesajlar verelim. birinci if blogu yanlissa baska kosullar test etmek icin else if komutundan yararlanacagiz.

< script>
var ortalama=56;
var mesaj="";
if(ortalama>=85){
mesaj="Pekiyi";
}else if(ortalama>=75){
mesaj="iyi";
}else if(ortalama>=55){
mesaj="Orta";
}else if(ortalama>=45){
mesaj="Gecer";
}else{
mesaj="Zayıf";
}
alert(mesaj);


Kod nasil calisiyor:
ilk once if kosulu test edilir sonuc true if blogu calisir, diger bloklar(else if,else) atlanir ve alert mesaji verilir
if kosulu yanlissa bir sonraki else if kosulu test edilir,aynen ustteki if gibi davranilir dogruysa gecerli blok calisir ve alertte kadar herseyi atlar.
Adim adim bunlar tekrarlanir,hic bir kosul dogru degilse en son else blogu her durumda çalısır
