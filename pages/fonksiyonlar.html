Fonksiyonlar

Sıklıkla senaryodaki birçok yerde benzer bir işlem yapmamız gerekir.
Örneğin, bir ziyaretçi oturum açtığında, oturumu kapattığında ve belki başka bir yerde hoş bir mesaj göstermemiz gerekir.
Fonksiyonlar programın ana “yapı taşlarıdır”. Kodun tekrar edilmeden birçok kez çağrılmasına izin verir.
Zaten fonksiyonlara dair bir çok örneklerini gördük alert(message) gibi. Ancak kendi fonksiyonlarımızı da oluşturabiliriz.

Fonksiyona Giriş

function showMessage() {
alert( 'Hello everyone!' );
}

Yeni fonksiyon adıyla çağrılabilir: showMessage().
Örneğin:
function showMessage() {
alert( 'Hello everyone!' );
}
showMessage();
showMessage();

Çağrı showMessage()işlevin kodunu yürütür. Burada mesajı iki kez göreceğiz. Bu örnek, işlevlerin temel amaçlarından birini açıkça göstermektedir: kod çoğaltmasını önlemek. Mesajı veya gösterilme biçimini değiştirmemiz gerekirse, kodu tek bir yerde değiştirmek yeterlidir: çıktıyı veren işlev.

Yerel değişkenler

Bir işlevin içinde bildirilen bir değişken yalnızca bu işlevin içinde görünür.
Örneğin:

function showMessage() {
let message = "Hello, I'm JavaScript!"; // local variable
alert( message );
}
showMessage(); // Hello, I'm JavaScript!
alert( message ); // <-- Error! değişken bir local değişkendir

Dış değişkenler

Bir işlev bir dış değişkene de erişebilir, örneğin:

let userName = 'John';
function showMessage() {
let message = 'Hello, ' + userName;
alert(message);
} showMessage(); // Hello, John

Parametreler

Parametreleri ( fonksiyon argümanları olarak da bilinir ) kullanarak fonksiyonlara keyfi veriler aktarabiliriz .
Aşağıdaki örnekte, işlevin iki parametresi vardır: fromve text.
function showMessage(from, text) { // argumanlar from ve text
alert(from + ': ' + text);
}
showMessage('Ann', 'Hello!'); (*)
showMessage('Ann', "What's up?"); (**)

İşlev satırlarda çağrıldığında (*)ve (**)verilen değerler yerel değişkenlere fromve text. Sonra işlev bunları kullanır.
